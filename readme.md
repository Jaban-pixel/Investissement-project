# Description
- K-CAPITAL(nom provisoire) est une plateforme qui permet de mettre en relation plusieurs investisseurs dans le but d'investir dans des projets commun. 
- ces investissements peuvent porter sur plusieurs domaines telles que le batimment, la securité et surtout les petits commerces.
# Objectifs
- Venir en aide aux petits commerces prometteurs ayant du mal à se retrouver financièrement en leurs donnons l'occasion de rencontrer des investisseurs sérieux.
- Garantir la sécurité des données des commercants ainsi que des investisseurs par une procédure de vérification de profil.
- Proposer un service d'accompagnements payant pour les utilisateurs ayant du mal avec les procédures de régularisation et les taches administratives.(à revoir)
- si possible garantir un système de transactions  en COBOL(juste pour le kiffe)
